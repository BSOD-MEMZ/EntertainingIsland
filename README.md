## 🚨 防巡堂警报

按下 `Ctrl+Shift+J` 全屏弹窗——**窗外有老师！**

- 自定义警报文字、热键
- 强调效果（主题色涟漪动画）
- 显示时长 / 语音播报 / 提示音效 / 窗口置顶
- 设置入口：ClassIsland 设置 → **提醒** → 窗外有老师警报

## 🛎️ 下课倒计时提醒

距下课 N 秒自动弹出提醒，样式 **一比一照搬 ClassIsland 内置「即将上课」**。

- 左右分栏：动态倒计时 + 下节课信息
- 自定义提前秒数、遮罩文字、详细文字
- 语音播报 / 强调效果 / 显示教师名称
- 内置 **预览** 按钮，方便调试
- 设置入口：ClassIsland 设置 → **提醒** → 下课倒计时提醒

## 📖 小说阅读器

把 txt 小说放到课表上滚屏阅读。

- 自动翻页（可调间隔）
- 显示页码和进度百分比
- `Ctrl+Shift+↑/↓` 翻页 · `Ctrl+Shift+Space` 暂停/继续
- 自动保存阅读进度

## 💬 口头禅记录

记录老师说了多少次「不要讲话」「安静」「看黑板」……

- 默认 `Ctrl+Shift+F1`~`F3` 三个预设
- 显示如「不要讲话 ×5  安静 ×3」
- 完全自定义口头禅文字和快捷键
- 每日自动清零

## 🔠 RSS 新闻

订阅 RSS/Atom 源，在课表上轮播显示。

- `Ctrl+←` / `Ctrl+→` 翻页
- `Ctrl+Shift+O` 打开当前链接
- 内置知乎日报、36氪、少数派等中文源

## 👤 头像课程表

以老师头像 + 科目图标展示当日课程表。

- 自定义头像—科目—教师三层映射
- 已上课程自动降饱和度 + 降低透明度
- 当前课程放大 + 入场动画
**警告：不建议为真人老师头像使用最低饱和度（黑白）效果**

## 🎲 点名器

随机抽人，但一点都不公平。

- 点击按钮弹出菜单：抽一人 / 抽两人
- 可拖拽屏幕浮窗（48×48 圆角方块）
- 结果同步 ClassIsland 通知
- 支持 txt 导入名单
- 隐藏爆率面板：`Ctrl+Shift+Alt+C`（加权 + 保底）

## ⚽ 体育赛事

实时比赛数据，来自 TheSportsDB。

- 预设 11 个主流联赛（NBA / 英超 / 西甲 / 德甲 等）
- **全部**模式轮播所有联赛
- 数据自动翻译为简体中文
- 详细 / 简洁两种显示模式
- `Ctrl+Shift+M` 切换显示模式

## 🔒 全局安全键

`Ctrl+Shift+K` —— 紧急情况下的救命稻草：
1. 立即消警
2. 一键隐藏/显示所有娱乐组件

## 🤖 ClassIsland 自动化

11 个自动化行动，支持在 ClassIsland 规则中按时间/状态触发：

| 类型 | 行动 |
|---|---|
| 全局组件 | 切换显隐 / 全部显示 / 全部隐藏 |
| 小说阅读器 | 暂停 / 继续 / 下一页 / 上一页 / 重新开始 |
| RSS 新闻 | 上一条 / 下一条 |
| 口头禅 | 清空记录 |

## 📦 安装

1. 下载 `.cipx` 插件包
2. 在 ClassIsland 中打开「插件」→「安装插件」→ 选择文件
3. 或者手动放入 `data\Plugins\entertainingisland.app\`

## 🔧 开发

```bash
# 构建
dotnet build -c Release

# 部署到本地 ClassIsland
.\deploy.ps1
```

## 📋 发布打包

> 建议先用 `dotnet publish -c Release`，再从 `publish/` 目录取文件。

打包为 `.cipx`（即 zip 改后缀），需包含：

- `manifest.yml` — 插件清单
- `icon.png` — 插件图标
- `EntertainingIsland.dll` — 主程序集
- `EntertainingIsland.deps.json` — 依赖清单
- `LibVLCSharp.dll` / `LibVLCSharp.Avalonia.dll` — VLC 依赖
- `runtimes/` — .NET 运行时文件
- `libvlc/` — VLC 原生库（可选）
- `README.md` — 说明文档（可选）

> ❌ 不需要包含 `.pdb` 调试符号文件。
